# Component-gen
Generate astro files with gun cli

# prerequisites
install [charm gum](https://github.com/charmbracelet/gum)

## How to use
### Executable: 
use chmod to execute the file
```sh
chmod +x commit.sh

# then execute the sript 
./commit.sh
```

### With alias! 
create an alias to execute the file anyware
```sh title:config.fish
# Fish example
alias commit = "sh path/to/file/commit.sh"

```
